# Richo-ML-projects
This repository contains multiple machine learning projects including a movie recommendation system, classification models, and clustering analysis. The projects utilize TensorFlow and Scikit-Learn with real-world datasets for predictive modeling and data exploration.

🚀 Machine Learning Projects

This repository contains several machine learning implementations including:

🎬 Recommendation System
  - Built using collaborative filtering with embedding
  - Dataset: MovieLens
  - Techniques: TensorFlow, similarity (cosine & dot product)
📊 Classification
- Models:
    - Decision Tree
    - Random Forest
- Includes hyperparameter tuning
- Saved models in .h5 format
🔍 Clustering
-Unsupervised learning approach
- Techniques:
    - K-Means / clustering model
    - PCA for dimensionality reduction
- Includes dataset and inverse transformation

How to Run

1. Open notebook using Google Colab / Jupyter
2. Install dependencies:
pip install -r requirements.txt
3. Run each notebook step by step

Purpose
This project aims to demonstrate practical implementation of machine learning models for recommendation, classification, and clustering problems.
